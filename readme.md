# dermitrack-supabase

Backend infrastructure for DermiTrack — PostgreSQL migrations, Edge Functions, and Supabase configuration.

## Local Development

```bash
supabase start          # Start local dev environment
supabase status         # Show local URLs and keys
supabase migration list # List applied migrations
supabase db push        # Apply pending migrations
```

See the root `CLAUDE.md` for full architecture documentation.
